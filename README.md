# Power BI Sales Analysis Dashboard

## Project Overview
This project is a 3-page interactive sales dashboard built using Microsoft Power BI.

## Report Pages

### 1. Sales Report Analysis
- Total Sales
- Profit Analysis
- Monthly Trends
- KPIs

### 2. State & Region Analysis
- Region-wise Sales
- State-level performance
- Regional comparison

### 3. Category & Sub-Category Analysis
- Category level sales
- Sub-category insights
- Profit distribution

## Tools Used
- Microsoft Power BI
- DAX
- Data Modeling
- Power Query
